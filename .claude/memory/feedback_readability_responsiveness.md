Always check readability before finalising any design.

**Why:** User added this as a hard rule after noticing text blending into dark backgrounds and cramped/missing padding across footer and sections.

**How to apply:** Run through all five checks before every output — especially on dark backgrounds, footers, and legal pages.

Mandatory checks:
- All text must have strong contrast against its background
- Never place text on a background where it becomes hard to read
- If text sits over a dark section or image, apply correct overlay or contrast treatment
- All sections must have proper padding and spacing — no elements sitting too close to edges
- Maintain strong spacing rhythm between headings, text, buttons, cards, and footer elements
- Footer content (legal links, copyright, developer credit) must always be clearly readable and properly spaced
- Legal links, copyright text, and developer credit must never blend into the background

Responsive rule:
- Every page and section must work at mobile, tablet, laptop, and large desktop
- No overlapping text, cut-off content, uneven padding, broken alignment, or horizontal overflow

Before final output, always review:
1. Colour contrast
2. Readability on every background
3. Spacing and padding
4. Mobile responsiveness
5. Tablet responsiveness
6. Desktop responsiveness

If text is hard to read → not finished.
If spacing feels weak or cramped → not finished.
If the layout breaks on smaller screens → not finished.
